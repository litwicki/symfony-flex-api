# Tavro Local Development

Build and deploy the API environment.

## Getting Started

    git clone git@github.com:zoadilack/tavro-api.git && cd tavro-api
    docker-compose up -d
    docker exec -it tavro_api /bin/sh /var/www/html/bin/init.sh
